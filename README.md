# homebrew-tap

Homebrew tap for [mguilhermetavares](https://github.com/mguilhermetavares) tools.

## Usage

```bash
brew tap mguilhermetavares/tap
brew install odin-writer
```

## Tools

| Tool | Description |
|------|-------------|
| [odin-writer](https://github.com/mguilhermetavares/odin-writer) | CLI that converts podcast episodes and YouTube videos into written articles for websites |
